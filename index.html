Project Structure

The FileUI project is organized into version-specific folders, each containing a complete implementation:

  • v001/ - Original BSP panel system with core functionality
  • v002/ - Reserved for future incremental improvements
  • v003/ - Enhanced panels with toolbar, markdown, and state management
  • v004/ - Modern IDE interface with modular ES6 architecture
